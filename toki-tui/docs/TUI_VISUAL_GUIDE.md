## Visual Elements Key

### Colors (in actual TUI):
- **Cyan**: Headers, titles
- **Green**: Running timer
- **Yellow**: Stopped timer, keyboard shortcuts, selected items
- **White**: Regular text
- **Gray**: Status messages
- **Bold Yellow**: Currently selected item in lists

### Interactive Elements:
- **Arrow keys (â†‘â†“)**: Navigate through lists (projects, activities, history)
- **Enter**: Confirm selection
- **Space**: Toggle timer on/off
- **Esc**: Cancel and return to main view
- **Tab**: Switch between Timer and History views
- **Single letters**: Quick actions (P, A, H, R, Q)

### Visual Feedback:
- **Timer state**: Color changes (Green=Running, Yellow=Stopped)
- **Selection**: Highlighted in bold yellow
- **Status bar**: Shows last action result
- **Project display**: Shows "(not running)" when stopped
